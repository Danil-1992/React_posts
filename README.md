# 📝 My React App
 
Приложение для просмотра и загрузки постов с JSONPlaceholder API.

## 🚀 Технологии

- **React 19** + **TypeScript**
- **Redux Toolkit** — управление состоянием
- **React Router v7** — навигация
- **Vite** — сборка
- **Feature-Sliced Design (FSD)** — архитектура
- **Jest 30** + **Testing Library** — тестирование
- **Zod** — валидация данных

## 📦 Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Danil-1992/React_posts.git
cd my-react-app

# 2. Установите зависимости
npm install

# 3. Запустите проект в режиме разработки
npm run dev

# 4. Запуск всех тестов
npm test
```
